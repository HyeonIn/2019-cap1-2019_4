### 1. 프로잭트 소개

Flex Ads는 회원제 오프라인 마켓에서 얼굴 인식을 이용하여 개인 또는 그룹에게 맞춤형 광고를 송출하는 프로젝트입니다. <br/>
얼굴 인식과 광고 추천 기능으로 나뉘며 모듈형 AI 슈퍼컴퓨터 'NDIVIA Jetson TX1와 TX2'를 이용하여 실시간으로 얼굴을 인식하고 회원의 정보를 추천 시스템과 연결합니다.<br/>
